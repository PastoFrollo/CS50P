# CS50P - Introduction to Programming with Python

Questo repository contiene le mie soluzioni personali ai problemi e agli esercizi del corso **CS50P** (Introduction to Programming with Python) offerto da **Harvard University** tramite la piattaforma edX.

Il corso copre i fondamenti della programmazione in Python: partendo dalle basi fino a concetti più avanzati come i test unitari: le espressioni regolari (regex) e la programmazione orientata agli oggetti (OOP).

---

## 🚀 Argomenti Trattati

- **Week 0:** Functions: Variables (Funzioni e Variabili)
- **Week 1:** Conditionals (Strutture di controllo condizionali: `match/case`)
- **Week 2:** Loops (`while`: `for`: iterazioni)
- **Week 3:** Exceptions (Gestione degli errori e delle eccezioni)
- **Week 4:** Libraries (Utilizzo di moduli integrati e pacchetti di terze parti via `pip`)
- **Week 5:** Unit Tests (Scrittura di test automatizzati con `pytest`)
- **Week 6:** File I/O (Lettura e scrittura di file CSV: immagini e file di testo)
- **Week 7:** Regular Expressions (Validazione e parsing di dati con le regex)
- **Week 8:** Object-Oriented Programming (Classi: metodi: proprietà e principi OOP)
- **Week 9:** Et Cetera (Funzioni avanzate: list comprehensions: generatori)

## 🛠️ Come Eseguire i Programmi

Per eseguire uno qualsiasi degli script presenti in questo repository: assicurati di avere Python installato sul tuo computer. Puoi clonare la cartella ed eseguire i file da terminale:

```bash
# Clona il repository
git clone [https://github.com/PastoFrollo/CS50P.git](https://github.com/PastoFrollo/CS50P.git)

# Entra nella cartella del corso
cd CS50P

# Entra nella cartella della settimana desiderata (es. 1_w)
cd "1_w"

# Esegui lo script (es. interpreter.py)
python interpreter.py
```
